# Introduction

This a web app, a team course project in SYSU.

# Requirement

- python3
- django 1.96
- PostgreSQL 9.5

# Run

run following command under mysite file

```
./manage.py runserver
```

visit `localhost:8000` to see the local website.

visit `localhost:8000/admin` to manage the website as super user.
